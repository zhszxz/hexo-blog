# Markdown语法速查

本文档整理常用Markdown语法，按「基础格式」「进阶样式」「实用功能」分类，简洁明了、可直接对照使用，适合快速查阅。

<!--more-->

# 一、基础文本格式（必学）

| 语法用途                 | Markdown语法                                                 | 效果预览                                               |
| ------------------------ | ------------------------------------------------------------ | ------------------------------------------------------ |
| 标题（1\-6级）           | \# 一级标题\#\# 二级标题\#\#\# 三级标题\#\#\#\# 四级标题\#\#\#\#\# 五级标题\#\#\#\#\#\# 六级标题 | 一级标题（最大）二级标题（次之）\.\.\.六级标题（最小） |
| 加粗                     | \*\*加粗文本\*\* 或 \_\_加粗文本\_\_                         | **加粗文本**                                           |
| 斜体                     | \*斜体文本\* 或 \_斜体文本\_                                 | *斜体文本*                                             |
| 加粗\+斜体               | \*\*\*加粗斜体文本\*\*\* 或 \_\_\_加粗斜体文本\_\_\_         | ***加粗斜体文本***                                     |
| 删除线                   | \~\~删除文本\~\~                                             | ~~删除文本~~                                           |
| 下划线（部分编辑器支持） | \&lt;u\&gt;下划线文本\&lt;/u\&gt;                            | 下划线文本                                             |
| 段落换行                 | 段落末尾加2个空格\+回车，或空一行                            | 两个段落（换行效果）                                   |
| 水平分隔线               | \-\-\- 或 \*\*\* 或 \_\_\_（单独一行）                       | 一条水平横线（分隔内容）                               |

# 二、列表格式

## 2\.1 无序列表

| 语法                                                   | 效果预览                            |
| ------------------------------------------------------ | ----------------------------------- |
| \- 列表项1\- 列表项2\- 列表项3（\- 可替换为 \* 或 \+） | - 列表项1<br>- 列表项2<br>- 列表项3 |

## 2\.2 有序列表

| 语法                                                  | 效果预览                               |
| ----------------------------------------------------- | -------------------------------------- |
| 1\. 列表项12\. 列表项23\. 列表项3（数字后必须加 \. ） | 1. 列表项1<br>2. 列表项2<br>3. 列表项3 |

## 2\.3 嵌套列表（无序列表\+有序列表）

| 语法                                                         | 效果预览                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| \- 父列表项1\&amp;nbsp;\&amp;nbsp;\&amp;nbsp;\&amp;nbsp;1\. 子列表项1\&amp;nbsp;\&amp;nbsp;\&amp;nbsp;\&amp;nbsp;2\. 子列表项2\- 父列表项2\&amp;nbsp;\&amp;nbsp;\&amp;nbsp;\&amp;nbsp;\- 子列表项3 | - 父列表项1<br><br>    1. 子列表项1<br>    2. 子列表项2<br>- 父列表项2<br>                <br>    - 子列表项3 |

注：嵌套时，子列表前加 4 个空格（或 1 个制表符），与父列表对齐。

# 三、引用格式

| 语法用途       | Markdown语法                                              | 效果预览                                                     |
| -------------- | --------------------------------------------------------- | ------------------------------------------------------------ |
| 普通引用       | \&gt; 引用文本                                            | 引用文本                                                     |
| 嵌套引用       | \&gt; 一级引用\&gt;\&gt; 二级引用\&gt;\&gt;\&gt; 三级引用 | <br>              一级引用<br>              <br>                二级引用<br>                三级引用 |
| 引用\+其他格式 | \&gt; \*\*加粗引用\*\*\&gt; \*斜体引用\*                  | **加粗引用***斜体引用*                                       |

# 四、代码格式

## 4\.1 行内代码

| 语法                                   | 效果预览                 |
| -------------------------------------- | ------------------------ |
| \`行内代码片段\`（反引号，键盘左上角） | 这里有一段`行内代码片段` |

## 4\.2 代码块（常用）

| 语法                                                         | 效果预览                                                     |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| \`\`\`语言类型代码内容\`\`\`（语言类型可选：python、java、html等，用于语法高亮） | `\# Python代码示例<br>print\(\&\#34;Hello Markdown\&\#34;\)` |
| 无语法高亮代码块\`\`\`普通代码内容\`\`\`                     | `普通代码内容，无语法高亮`                                   |

# 五、链接与图片

## 5\.1 超链接

| 语法类型                     | Markdown语法                                    | 效果预览                                             |
| ---------------------------- | ----------------------------------------------- | ---------------------------------------------------- |
| 普通链接                     | \[链接文本\]\(链接地址\)                        | [链接文本](https://www.example.com)（点击跳转）      |
| 带标题的链接（鼠标悬浮显示） | \[链接文本\]\(链接地址 \&\#34;链接标题\&\#34;\) | [链接文本](https://www.example.com)（悬浮显示标题）  |
| 直接显示链接（自动识别）     | \&lt;链接地址\&gt;                              | [https://www\.example\.com](https://www.example.com) |

## 5\.2 插入图片

| 语法类型 | Markdown语法                                            | 说明                                              |
| -------- | ------------------------------------------------------- | ------------------------------------------------- |
| 本地图片 | \!\[图片alt文本\]\(本地图片路径\)                       | 路径可为相对路径（如 \./images/1\.jpg）或绝对路径 |
| 网络图片 | \!\[图片alt文本\]\(网络图片URL \&\#34;图片标题\&\#34;\) | alt文本：图片加载失败时显示；标题：鼠标悬浮显示   |

示例：\!\[Markdown图标\]\(https://example\.com/markdown\.png \&\#34;Markdown图标\&\#34;\)

# 六、表格（常用）

| Markdown语法                                                 | 效果预览 |       |       |
| ------------------------------------------------------------ | -------- | ----- | ----- |
| \| 表头1 \| 表头2 \| 表头3 \|\| \-\-\- \| \-\-\- \| \-\-\- \|\| 内容1 \| 内容2 \| 内容3 \|\| 内容4 \| 内容5 \| 内容6 \|（\-\-\- 用于分隔表头和内容，可省略） | 表头1    | 表头2 | 表头3 |
| 内容1                                                        | 内容2    | 内容3 |       |
| 内容4                                                        | 内容5    | 内容6 |       |

注：表格对齐（可选）：\| \-\-\-: \| :\-\-\- \| :\-\-\-: \| （分别对应右对齐、左对齐、居中对齐）。

# 七、进阶实用功能（部分编辑器支持）

| 功能             | Markdown语法                           | 说明                                             |
| ---------------- | -------------------------------------- | ------------------------------------------------ |
| 任务列表         | \- \[x\] 已完成任务\- \[ \] 未完成任务 | x 为小写，括号内空格不可少                       |
| 脚注             | 文本内容\[^1\]\[^1\]: 脚注说明文字     | 点击脚注标记可跳转至说明                         |
| 目录（自动生成） | \[TOC\]                                | 部分编辑器（如Typora）支持，自动提取标题生成目录 |
| 公式（LaTeX）    | 行内公式：$E=mc^2$块级公式：$$E=mc^2$$ | 用于插入数学公式，需编辑器支持LaTeX              |

# 八、注意事项

- Markdown语法对空格、换行敏感，部分语法需严格遵循格式（如标题后加空格、列表前加空格）。

- 不同编辑器（Typora、VS Code、GitHub等）对语法的支持略有差异，优先使用通用语法。

- 复杂样式（如颜色、字体大小）可结合HTML标签（如 \&lt;font color=\&\#34;red\&\#34;\&gt;红色文本\&lt;/font\&gt;），但兼容性较差。
